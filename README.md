# aspire-vite-react

Example Vite/React web app with [Aspire](https://aspire.dev/)

Includes two web apps, one using the `pnpm` package manager and the other using the `npm` package manager.

## Prerequisites

Requires .NET 10 SDK or later.

Install the Aspire CLI by following the instructions at <https://aspire.dev/get-started/install-cli/>

Then run

```bash
aspire run
```
